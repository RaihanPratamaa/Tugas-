Url 
    A URL is another word for a web address.

A URL can be composed of words (e.g.
w3schools.com), or an Internet Protocol 
(IP) address (e.g. 192.68.20.50).

Most people enter the name when surfing,
because names are easier to remember than
numbers.

A Uniform Resource Locator (URL) is used to
address a document (or other data) on the web.
